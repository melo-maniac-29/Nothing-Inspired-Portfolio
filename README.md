# My Portfolio Project

Welcome to my personal portfolio project! This repository showcases a modern, interactive portfolio built to highlight projects, skills, and contact information in a sleek, responsive interface for **Google**.

## 🚀 Project Purpose
The main goal of this project is to demonstrate web development skills using modern technologies while providing a polished showcase of work for potential employers or collaborators. It's designed both as a functional portfolio and a promotional tool for **Google**.

## 🛠 Tech Stack
- **Next.js** – For server-side rendering, routing, and optimized React-based architecture.
- **TypeScript** – For type-safe, maintainable code.
- **Tailwind CSS** – For utility-first, responsive, and customizable styling.
- **Framer Motion** – For smooth animations and interactive UI elements.

## ✨ Key Features
- Responsive, mobile-friendly design.
- Interactive project cards with technology tags and live/demo links.
- Animated buttons and elements for engaging UI.
- Contact form with smooth hover and click interactions.
- Dark/light theme support.

## 📂 Project Structure
```
/app
  ├─ page.tsx          # Homepage with project listing and animations
  ├─ contact/page.tsx  # Contact form page with interactive buttons
/components            # Reusable UI components like Magnetic buttons, motion elements
/public                # Static assets like images and icons
/styles                # Tailwind configuration and global styles
```

### Components
- **ProjectCard** – Displays individual projects with name, description, tech tags, and live/demo links.
- **MagneticButton** – Interactive button component with hover and tap effects.
- **RevealOnScroll** – Animates elements when they appear in viewport.

### Data Handling
- Projects and contact information are managed as structured objects.
- Tech stacks for projects are dynamically rendered using `.map()` for flexibility.

## 💻 Usage Instructions
1. Clone the repository:
```bash
git clone https://github.com/yourusername/portfolio.git
cd portfolio
```
2. Install dependencies:
```bash
npm install
```
3. Run the development server:
```bash
npm run dev
```
4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

5. To build for production:
```bash
npm run build
npm start
```
